扫雷游戏
等级：初级（10*10）
      中级（15*15）
      高级（20*20）
1. 游戏者自行选择难度等级，点击“开始游戏”进行游戏；点击左键，若点击位置为雷所在位置，游戏结束，弹出对话框询问是否开始新的游戏，确定则开始新游戏，取消则关闭游戏窗口，若在雷的位置点击右键，则表示对雷进行了标记；若点击位置不是雷所在位置，则显示该方格周围雷的数量；正确标记所有地雷并打开所有非地雷格子，赢得游戏，弹出对话框询问是否再玩一次；

2. 游戏过程中，可以提示剩余雷的总数，并对游戏进行计时，游戏结束计时停止；若当前点击方格周围（向上一行向左一列，向下两行向右两列范围内）存在相邻位置没有地雷的方格，则将该方格背景设置为白色；

3.点击重新开始，则开始一轮新游戏，并重新开始计时
